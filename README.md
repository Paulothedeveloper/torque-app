# Torque — Plataforma all-in-one pra oficina mecânica

![status](https://img.shields.io/badge/status-NO_AR-blue) &nbsp; por **Paulo Adriel** · [Paulocodex](https://paulocodex.com)

Torque é o sistema all-in-one pra oficina mecânica: ordens de serviço, clientes, estoque, orçamentos e financeiro num painel só — do orçamento à entrega do carro. Menos papel, mais controle. SaaS por assinatura.

## O que faz

- Ordens de serviço do início ao fim
- Cadastro de clientes e histórico
- Estoque e orçamentos
- Financeiro num painel só
- Planos por tamanho da oficina

## Status

**No ar** — em produção, evoluindo.
